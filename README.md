# React burger

Сайт космической бургерной. Можно собирать бургеры, создавать заказы, отслеживать текущее состояние заказа. 


Ссылка на проект: https://simajiawei.github.io/react-burger/

## Технологии

- react
- redux
- react-dnd
- cypress (тесты)
- websocket

## Установка
```bash
npm i
```

## Запуск

```bash
npm run start
```

## Тесты

```bash
npm run test 
```

e2e тесты:
```bash
npm run cypress:open
```

## Деплой

```bash
npm run deploy
```
